# 人数制限SNS

グループの上限を設けることで、密度の高いつながりを目指したSNS  

## 特徴
- グループごとに参加人数の上限あり
- 投稿・コメント・投票機能
- グループ内での履歴管理（ライブラリ）

## 概要
- ログイン（秘密鍵方式）
- 投稿・コメント
- タグ付きライブラリ
- 投票 / 投票履歴

## 使用技術
- Frontend: React
- Backend: Node.js 
- Database: SQL  
- Deployment: ConoHa VPS
